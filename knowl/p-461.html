  • Reduccible Error: This is error that we can eliminate by using a better model for the relationship. It is due in this example to us using a sample.

  • Irreduccible Error, \(E\text{:}\) This is the underlying error we introduced when we simulated the data. In real data it is due to factors not included in the model. This cannot be removed and our goal is instead to understand this term.
