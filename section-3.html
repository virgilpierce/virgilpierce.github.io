Subsection 1.3.2 Jupyter Notebooks

The textbook is compiled from Jupyter notebooks used as notes and worksheets in class. We will use github to distribute these here (and below). I will accept and be grateful for typos and other mistakes you find in this text.

Subsection 1.3.3 Github

There are two things we need to address in our class that we will use Github to get around.

Problem 1: I need to get you the Jupyter notebooks and datasets we will use for class. Both of these are in the Github repository for our class available at this link . I have made it a public repository so it should be easy to find. You have two choices for using this link, you can download individual files from the repository, or you can set up your Github account and then download or clone the repository to your own computer to then open the files in Jupyter. I will accept, and be greatful for, commits back to this repository if you find typos or other things that need to be change. You can also use the "Issues" tab in the Github website to let me know about problems you have seen.

Problem 2: You will need to be sharing files with your team and also controlling the versions you are working on. Github is used by many companies for controlling the versioning of their files, and that is how we will use it. This is a great skill to be able to talk to employers about that not everyone they interview will have. You should create a Github account. We will cover in class how your team will set up your project and the workflow you should follow.
